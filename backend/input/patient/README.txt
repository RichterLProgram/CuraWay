Medical Reports – Input Folder
==============================

Drop patient reports here in common formats:

Always supported (no extra packages):
  • .txt, .md, .rtf, .xml – text files
  • .json                – pre-extracted IDP JSON
  • .csv                 – tables (CSV)

Optional (pip install -r requirements.txt):
  • .docx – Word (python-docx)
  • .xlsx – Excel (openpyxl)
  • .xls  – Excel legacy (xlrd)
  • .pdf  – PDF (pypdf + optional OCR)
  • .odt  – OpenDocument Text (odfpy)
  • .png/.jpg – images (OCR)

Files are processed alphabetically.
