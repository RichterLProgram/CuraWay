# Patient Flow (CancerCompass)

## Run
```powershell
cd backend
python .\flow_patient\run.py
```

## Input
Place patient reports in `backend/input/`.

## Output
`backend/output/`:
- `trial_matching_result.json`
- `trial_matching_top3.json`
- `trial_matching_trace.json`
- `patient_api.json`
- `demand_points_from_patients.json`
